Abstract
Documentation: 
Pénztárnyitás bizonylat, napi forgalmi jelentés, pénztárjelentés, pénzmozgás bizonylat és minden egyéb, nem adóügyi bizonylat fogadását megvalósító szolgáltatás kérés típusa.
The request type of the service which receives e-cash register open balance report, daily cashflow report, e-cash register report, cash flow document and all other non tax documents.
