Abstract
Documentation: 
Amennyiben a dokumentum on-line beküldés valamilyen okból megvalósult, a kapcsolat helyrállása után kell beküldeni, ebben az esetben a mezőben jelezni kell, hogy a dokumentum off-line került beküldésre.
If for some reason the document was submitted on-line, it should be submitted after the connection has been established, in which case the field should indicate that the document was submitted off-line.
